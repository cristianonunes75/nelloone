## 4. VOCABULÁRIO PROPRIETÁRIO

### Substituições aplicadas em toda a landing:

| Antes | Depois |
|-------|--------|
| teste | dimensão |
| perfil | leitura |
| resultado | mapa |
| avaliação | jornada |
| relatório | síntese |
| testes reconhecidos | dimensões integradas |
| 7 testes profissionais | 7 dimensões integradas |
| Resultado Final | Síntese Identity |
| PDF Personalizado | Síntese Viva |
| Mapa NELLO IDENTITY completo | Síntese Identity completa |

### Termos removidos:
- testes psicológicos
- avaliações
- perfis
- questionários

### Termos introduzidos:
- dimensões humanas
- leitura integrada
- arquitetura comportamental
- mapa da identidade
- síntese viva
